## [0.6.0] — 2026-03-17

**Theme: Logger control** — Four new options giving teams precise control over what appears in terminal and log files.

### Added

- `logIncoming: false` — hide `>>` arrival lines, keep only `<<` response lines
- `shortRequestId: true` — show last 8 chars of request ID on log lines only
- `logClientIp: true` — log client IP on each response line
- `lineColor: 'level'` — colour log lines by response status instead of HTTP method
